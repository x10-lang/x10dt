7  Statements and expressions

X10 supports the standard set of primitive operations (assignment, classcasts) and sequential control constructs (conditionals, looping, method invocation, exception raising/catching) etc.
Place casts   The programmer may use the standard classcast mechanism (§ ) to cast a value to a located type. A BadPlaceException is thrown if the value is not of the given type. This is the only language construct that throws a BadPlaceException.
