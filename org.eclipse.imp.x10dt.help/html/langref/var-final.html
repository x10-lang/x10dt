1  Immutable variables

An immutable variable satisfies two conditions:
  • it can be assigned to at most once,
  • it must be assigned to before use.
X10 follows Java language rules in this respect [,§ 4.5.4,8.3.1.2,16]. Briefly, the compiler must undertake a specific analysis to statically guarantee the two properties above.
Immutable local variables and fields are defined by the val keyword. Elements of value arrays are also immutable.
