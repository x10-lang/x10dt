X10DT: Eclipse tools for the X10 language

X10DT, or "X10 Development Toolkit," is the set of Eclipse plug-ins that lets you develop X10 code in Eclipse. This includes editing, syntax highlighting, compiling, running, and (soon) debugging. The X10DT makes it easy to be productive writing and executing your X10 application.
