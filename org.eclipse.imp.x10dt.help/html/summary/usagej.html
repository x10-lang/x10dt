Usage for Java backend

Usage: x10 [options]  [arg0 arg1 ...]
where [options] includes:
    -t -time                 display execution time
    -v -verbose --verbose    print verbose debugging information
    -h -help --help          print this message
    -mx                set max heap size, e.g., -mx 200M
    -D=         set system property  to 
    -classpath         search path for class files
    -libpath           search path for native libraries
    -config            (IGNORED) read configuration from etc\\.cfg
    -dev                     developer mode (use unpackaged X10 libraries)
    -J                  [USE WITH CAUTION] pass  directly to java.
        e.g., use -J-verbose to make java execution verbose.
