Running your X10 Application - C++ backend

This section assumes you have already built your X10 Application with the C++ back-end. See Creating an X10 project with the C++ back-end for more information on doing this.

Launching

Once a project has been successfully built, one can create a new launch configuration for the X10 project with the C++ back-end.

To access the launch configuration dialog, select Run > Run Configurations...

Then create a new launch configuration for "X10 Application (C++ back-end)"
