in is a binary operator, definable in §. It is conventionally used for checking containment.
The built-in type Region provides in, testing whether a Point is in the region: 
assert [3] in 1..10;
assert !([10] in 1..3);
Other types can provide them as well: 
class Cont {
   operator this in (Int) = true;
   operator (String) in this = false;
   static operator (Cont) in (b:Boolean) = b;
   static def example() {
      val c:Cont = new Cont();
      assert c in 4 && !(ödd" in c) && (c in true);
   }
}
