5  Distributed Arrays

Distributed arrays, instances of DistArray[T], are very much like Arrays, except that they distribute information among multiple Places according to a Dist value passed in as a constructor argument. For example, the following code creates a distributed array holding a thousand cells, each initialized to 0.0, distributed via a block distribution over all places. 
val R <: Region = 1..1000;
val D <: Dist = Dist.makeBlock(R);
val da <: DistArray[Float] = DistArray.make[Float](D, (Point(1))=>0.0f);
