What's New in 2.5.2

Here are descriptions of some of the more interesting or significant changes made in X10 and X10DT for the 2.5.2 release. They are grouped into:

  • X10 Language
  • X10 Development Toolkit
  • Managed X10
  • Native X10
  • Additional Notes
X10 Language and Class Libraries
X10 Runtime moved to x10.xrx package The implementation of the X10 Runtime was moved from the x10.lang package to a new x10.xrx package. Unlike x10.lang, this package is not auto-imported into all programs. Programs that make calls to x10.xrx.Runtime (or other impacted classes) will need to be updated to import the used XRX classes.
