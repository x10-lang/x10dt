What's New in 2.3.1

Here are descriptions of some of the more interesting or significant changes made to the X10 Tooling Platform for the 2.3.1 release.

X10 Development Toolkit
Simplified remote build and launch

X10DT 2.3.1 introduces a simplified and streamlined mechanism for configuring remote build and launch. In the new system, a "X10 Remote Compilation" configuration is created via the "Run Configurations" menu of X10 DT that specifies the properties of the remote system. Once the Remote Compilation is specified, X10 programs built on the remote host can simply be launched via the normal "Run Configuration" mechanism of Eclipse.

A single X10 Remote Compilation configuration can be shared by multiple projects in the workspace.

Compatability with Eclispe 4.2

Although the pre-built X10DT continues to be based on Eclipse 3.7.2, it is now possible to install X10DT into a Eclipse 4.2/PTP 6 base.
