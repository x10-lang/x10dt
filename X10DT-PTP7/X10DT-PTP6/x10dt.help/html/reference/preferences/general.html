General Preferences

Indicate your preferences for the General settings on the X10 preference page.

To modify how your X10DT operates, bring up the X10 preferences page.

  • On most platforms, Window > Preferences.
  • On Mac, Eclipse > Preferences.

The X10DT preferences show when values have been modified, and show at what level the preferences are set - See the main X10 preferences page, shown above, for more information. As you change a preference setting, for example, its color changes to red indicating its value has changed.

Disable editor features (hyperlinking, hover help, etc.) for performance scalability

Because of performance issues with the X10 compiler front-end, we've added a preference to improve editor performance when working on large source files. When this preference is turned on, certain editor features like hyperlinking and hover help are effectively disabled, by bypassing the more expensive front-end processing phases like name resolution and type-checking. This will also effectively disable some of the content assist. Basic editor functionality, such as syntax highlighting, source code folding, and outlining continue to function as usual. Likewise, all non-editor functionality, such as building and launching, will also behave normally in this mode.
