Chapter 17
Annotations

X10 provides an an annotation system system for to allow the compiler to be extended with new static analyses and new transformations.
Annotations are interface types that decorate the abstract syntax tree of an X10 program. The X10 type-checker ensures that an annotation is a legal interface type. In X10, interfaces may declare both methods and properties. Therefore, like any interface type, an annotation may instantiate one or more of its interface's properties.
