Short-lived variables are introduced by local variables declarations, as described in §5. Local variables may be declared only within a block statement (§). The scope of a local variable declaration is the statement itself and the subsequent statements in the block. Variables declared in such statements shadow variables of the same name declared elsewhere. 
  if (a > 1) {
     val b = a/2;
     var c : Int = 0;
     // b and c are defined here
  }
  // b and c are not defined here.
A local variable of a given name, say x, cannot shadow another local variable or parameter named x unless there is an intervening method or closure declaration, or unless the inner x is declared inside an async or at statement and the outer variable is declared outside of that.
